[검색 · 앱 실행 (마우스 대체 1순위)]
⌘ + Space : 앱 실행·파일·계산·설정까지 즉시 검색
⌘ + Tab : 실행 중인 앱 빠르게 전환
⌘ + ` : 같은 앱 안에서 창 전환

[창 · 앱 정리 (트랙패드 의존도 급감)]
⌘ + W : 현재 창 닫기
⌘ + Q : 앱 완전 종료
⌘ + M : 창 최소화
⌘ + H : 현재 앱 숨기기
⌘ + Option + H : 현재 앱만 남기고 전부 숨기기
⌘ + Option + Esc : 앱 강제 종료

[파일 관리 (Finder 클릭 제거)]
⌘ + N : 새 Finder 창
⌘ + Shift + N : 새 폴더 생성
⌘ + Delete : 파일 휴지통 이동
⌘ + Shift + Delete : 휴지통 비우기
Space : 파일 미리보기 (열지 않고 내용 확인)
⌘ + Shift + G : 경로 직접 입력 이동
⌘ + ↑ : 상위 폴더 이동
⌘ + ↓ : 선택 항목 열기

[텍스트 이동 · 선택 (마우스 드래그 제거)]
Option + ← / → : 단어 단위 이동
⌘ + ← / → : 줄 맨 앞 / 끝 이동
⌘ + ↑ / ↓ : 문서 맨 위 / 아래 이동
Shift + ← / → : 문자 단위 선택
Option + Shift + ← / → : 단어 단위 선택
⌘ + Shift + ← / → : 줄 전체 선택

[화면 · 작업 전환]
Control + ↑ : 미션 컨트롤 (열린 창 한눈에)
Control + ← / → : 데스크톱 이동
⌘ + Control + F : 전체 화면 전환

[캡처 · 기록 (보고·공유 필수)]
⌘ + Shift + 4 : 선택 영역 캡처
⌘ + Shift + 5 : 캡처·화면 녹화 패널

[시스템 즉시 제어]
Control + ⌘ + Q : 즉시 화면 잠금
⌘ + , : 앱 환경설정 바로 열기
⌘ + Shift + . : 숨김 파일 표시/숨김

[기본 편집]

⌘ + X : 잘라내기
⌘ + Z : 실행 취소
⌘ + Shift + Z : 실행 취소 되돌리기
⌘ + A : 전체 선택
⌘ + S : 저장
⌘ + P : 인쇄
⌘ + W : 현재 창 닫기
⌘ + Q : 앱 완전 종료
⌘ + , : 앱 환경설정 열기

[앱 · 창 전환]
⌘ + Tab : 실행 중인 앱 전환
⌘ + ` : 같은 앱 내 창 전환
⌘ + H : 현재 앱 숨기기
⌘ + Option + H : 현재 앱 제외 전부 숨기기
⌘ + M : 창 최소화
⌘ + Option + Esc : 강제 종료 창 열기

[Finder (파일 관리)]
⌘ + N : 새 Finder 창
⌘ + Shift + N : 새 폴더 생성
⌘ + Delete : 휴지통으로 이동
⌘ + Shift + Delete : 휴지통 비우기
Space : 파일 미리보기 (Quick Look)
⌘ + 1 : 아이콘 보기
⌘ + 2 : 목록 보기
⌘ + 3 : 컬럼 보기
⌘ + 4 : 갤러리 보기
⌘ + I : 파일/폴더 정보 보기
⌘ + Shift + G : 경로로 이동
⌘ + ↑ : 상위 폴더 이동
⌘ + ↓ : 선택 항목 열기

[스크린샷 · 화면 녹화]
⌘ + Shift + 3 : 전체 화면 캡처
⌘ + Shift + 4 : 선택 영역 캡처
⌘ + Shift + 4 + Space : 선택 창 캡처
⌘ + Shift + 5 : 캡처·화면 녹화 패널
⌘ + Shift + 6 : 터치바 캡처 (지원 모델)

[텍스트 편집]
Option + ← / → : 단어 단위 이동
⌘ + ← / → : 줄 맨 앞 / 끝 이동
⌘ + ↑ / ↓ : 문서 맨 위 / 아래 이동
Shift + ← / → : 문자 단위 선택
Option + Shift + ← / → : 단어 단위 선택
⌘ + Shift + ← / → : 줄 전체 선택

[미션 컨트롤 · 데스크톱]
Control + ↑ : 미션 컨트롤 열기
Control + ↓ : 앱 창 보기
Control + ← / → : 데스크톱 이동
F3 : 미션 컨트롤 (기본 설정)
F11 : 데스크톱 보기

[시스템 · 검색]
⌘ + Space : Spotlight 검색
⌘ + Option + Space : Finder 검색
⌘ + Control + Space : 이모지·기호 입력
Control + ⌘ + Q : 화면 즉시 잠금

[사운드 · 밝기]
F1 / F2 : 화면 밝기 감소 / 증가
F10 : 음소거
F11 / F12 : 볼륨 감소 / 증가
Option + 밝기 키 : 밝기 세밀 조절
Option + 볼륨 키 : 볼륨 세밀 조절

[전원]
Control + ⌘ + 전원 버튼 : 강제 재시작
⌘ + Option + 전원 버튼 : 잠자기

[브라우저 공통]
⌘ + T : 새 탭 열기
⌘ + W : 탭 닫기
⌘ + Shift + T : 닫은 탭 다시 열기
⌘ + L : 주소창 이동
⌘ + R : 새로고침
⌘ + Option + I : 개발자 도구 (허용 시)

[기타 유용]
⌘ + Option + D : Dock 숨김/표시
⌘ + Control + F : 전체 화면 전환
⌘ + Shift + . : 숨김 파일 표시/숨김
